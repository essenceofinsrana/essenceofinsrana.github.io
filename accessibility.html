Accessibility Statement

Last Updated: 20 Jun 2026

This website has been developed with the aim of providing an accessible and user-friendly experience for all visitors, including persons using assistive technologies and alternative input methods.

1. Accessibility Features

Several accessibility-oriented features and practices have been incorporated throughout the website, including:

  • Use of semantic HTML elements to improve structure and navigation
  • Responsive design for compatibility across desktops, tablets, and mobile devices
  • Keyboard-accessible interactive components, including image galleries and dialog windows
  • Focus management within modal/lightbox interfaces
  • Alternative text for meaningful images
  • ARIA labels and accessibility attributes for navigation and interactive elements
  • Structured headings to improve readability and screen reader navigation
  • Readable typography and scalable layouts
  • Touch-friendly controls for mobile and touchscreen users
  • Accessible navigation menus and structured content sections
  • Support for keyboard navigation using standard keys such as Tab, Escape, and Arrow keys

Improving accessibility features in this website is an ongoing effort. While care has been taken to improve accessibility, the website may not yet fully conform to all WCAG 2.2 AA requirements.

We remain committed to improving accessibility and usability over time. Feedback regarding accessibility issues or suggestions for improvement is always welcome.

2. Contact Us

If you encounter any difficulty while using this website, please contact us through the Contact page so that we may review the issue and consider appropriate improvements.
